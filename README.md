# PLAYERIQ – AI Sports Performance Analytics

PLAYERIQ is a sports analytics dashboard that uses machine learning to predict player performance based on their statistics. I built this project to explore how AI can turn raw sports data into meaningful insights.

## Features

- Upload a CSV file with player statistics
- Predict player performance using machine learning
- Compare two players
- View AI-generated player rankings
- Explore interactive charts and visualizations
- Get simple performance insights

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- NumPy

## How It Works

The app reads a sports dataset, processes the player statistics, and calculates a performance score. A Linear Regression model then predicts player performance using features such as points, assists, rebounds, minutes played, and recent form.

The results are displayed in an interactive dashboard where users can explore predictions, compare players, and view rankings.
